# JobSearchPortal
## Framework and Language used
* Springboot
* Java

## Project Summary
* This is a SpringBoot-based project for managing jobs, with endpoints for adding, retrieving, updating, and deleting user information.

## user attributes
* id
* title
* description
* location
* salary
* companyName
* employerName
* jobType

## controller
* JobController

## Service
* JobService

## Model
* Job
* JobType(Enum)

## Repository
* JobRepository
* Used CrudRepository and Custom Query method

## Data Structure used
* ArrayList
